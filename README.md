# GitHub API Image Upload (C# Console App)

This project is a C# console application developed as part of an internship assessment.
## Features
- Uploads an image file from local system to a GitHub repository
- Uses GitHub REST API
- Implements authentication using GitHub Personal Access Token
- Converts binary image data to Base64 format before upload
## Technologies Used
- C#
- .NET Console Application
- GitHub REST API
## Note
The Personal Access Token used during development is not included in this repository for security reasons.
